2) внести 250
POST http://localhost:8081/api/v1/wallet
{
    "walletId" : "11111111-1111-1111-1111-111111111111",
    "operationType" : "DEPOSIT",
    "amount" : 250.50
}

3) вычесть 100
POST http://localhost:8081/api/v1/wallet
{
    "walletId" : "11111111-1111-1111-1111-111111111111",
    "operationType" : "WITHDRAW",
    "amount" : 100.00
}
